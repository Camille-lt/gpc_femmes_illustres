📖 **GPC - Femmes Illustres
🌟 **Présentation**
**GPC - Femmes Illustres** est un projet dédié à la mise en avant de **figures féminines historiques et contemporaines ayant marqué l'histoire, les sciences, les arts et la politique.**
L'objectif est de lutter contre l'invisibilisation des femmes en proposant une plateforme interactive et éducative.

Ce projet a été réalisé dans le cadre de ma formation à Ada Tech School, mettant l'accent sur le développement d'une interface utilisateur intuitive et la gestion dynamique de contenus.

🚀 **Stack Technique**
- Front-end : HTML5 / CSS3 (Flexbox & Grid pour le layout)
- Dynamisme : JavaScript (ES6+)
- Mapping : Intégration de l'API [Leaflet.js / Google Maps API] pour la cartographie interactive.
- Données Géo : Manipulation de coordonnées GPS (Latitude/Longitude) associées aux fiches biographiques.
- Architecture : Approche Component-based pour la structure
- Versionning : Git / GitHub

✨ **Fonctionnalités**
- Galerie Interactive : Présentation de portraits de femmes illustres avec des fiches détaillées.
- Filtres & Recherche : Possibilité de naviguer parmi les profils par domaine (Sciences, Art, Engagement, etc.).
- Responsive Design : Consultation fluide sur mobile, tablette et desktop.
- Accessibilité : Respect des normes sémantiques HTML pour un accès inclusif.
- Géolocalisation Historique : Visualisation interactive des lieux de vie ou des accomplissements majeurs de chaque femme via une carte intégrée.
- Marqueurs Dynamiques : Un marqueur personnalisé apparaît sur la map pour situer précisément le parcours de la femme illustre sélectionnée.

🎨 **Design**
Le design du projet a été pensé pour être à la fois sobre et percutant, mettant en valeur l'iconographie et les récits de vie.
Une attention particulière a été portée à la typographie et au contraste pour garantir une lecture agréable.
